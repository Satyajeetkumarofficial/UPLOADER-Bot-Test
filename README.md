# Telegram URL Uploader Bot

A powerful Telegram bot to upload files from URLs including:

- YouTube
- Instagram
- Facebook
- Mediafire
- Google Drive
- Direct Links
- (Terabox coming soon)

## Features

- 4GB upload limit for free users
- 20GB+ for premium users
- MongoDB-ready (can be added easily)
- Log channel support
- Easy to deploy on Koyeb
